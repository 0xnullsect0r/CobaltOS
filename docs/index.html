Why CobaltOS

Everything your Chromebook deserves

Built for the hardware you already own, without the Google dependency.

⚡

Blazing-fast boot

Desktop in under 10 seconds on eMMC thanks to systemd-boot, zram-generator, and a hand-tuned Debian base — no bloat.

🖥️

COSMIC Desktop

The brand-new Wayland-native desktop by System76. Built entirely in Rust with the iced framework — modern, beautiful, and snappy.

💻

Chromebook-native hardware

Keyboard remapping, audio UCM profiles, touchpad gestures, and HiDPI scaling configured automatically for your exact Chromebook model on first boot.

🦀

Rust-powered tooling

ripgrep, fd, eza, bat, zoxide, starship and more — pre-installed and aliased for a modern shell experience.

🛠️

Friendly installer

cobalt-installer guides you step-by-step with automatic hardware detection, UEFI firmware check, and auto-partitioning. Choose ext4 or btrfs (with subvolumes + zstd compression). GUI and TUI modes available.

🔒

Privacy-respecting

No Google account. No telemetry. No cloud lock-in. Debian Stable base with Flatpak for sandboxed apps from Flathub. Privacy preferences configured during setup — analytics and location access off by default.

📦

Huge software library

Full apt package ecosystem plus Flatpak with Flathub enabled by default. Install anything from VS Code to Blender in seconds. Welcome wizard installs your chosen browser and apps on first login.

🔄

Smart auto-updates

cobalt-update handles apt and Flatpak updates with GUI tray notifications and safe rollback on failure. A systemd timer checks every 6 hours — only notifying when new packages arrive.

✨

Polished first-run experience

cobalt-oobe walks you through timezone, privacy, and account setup before the desktop loads. cobalt-welcome then handles appearance, locale, browser choice, and app recommendations.

🐧

Full Linux compatibility

Mainline LTS kernel with Chromebook-specific patches. Every standard Linux application, developer tool, and game works as expected.

🔍 Is my Chromebook supported?

CobaltOS supports all Chromebooks compatible with MrChromebox's UEFI Full ROM firmware — covering most Chromebooks from 2012 onwards across Intel and AMD platforms.

How it works

From Chromebook to Linux in 5 phases

The full process takes 30–60 minutes. Follow the step-by-step guide.

1

Download & Flash

Get the ISO and write it to a USB drive with Rufus, Etcher, or dd.

2

Developer Mode

Enable ChromeOS Developer Mode to unlock firmware access.

3

Remove Write Protect

Remove the WP screw or disconnect the battery to allow firmware flashing.

4

Flash UEFI Firmware

Run MrChromebox's script to replace the factory firmware with UEFI.

5

Install CobaltOS

Boot from USB and follow the friendly guided installer.
